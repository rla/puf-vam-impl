-----------------------------------
- VAM -              Ein System zur
Visualisierung Abstrakter Maschinen
-----------------------------------

Starten:
  java.bat (Windows)
  oder
  java.sh  (Linux)
  oder
  java -jar vam.jar

Programm auswählen:
  "Program"-Button
  oder
  Menü "VAM -> Open Program ..."

Maschine auswählen:
  Menü "VAM -> Select Machine"
  (Auswahl der Maschine erfolgt auch automatisch über Dateiendung)

Verzeichnisse:
  vam.jar : Java-Klassen und Quellen
  examples: enthält Beispiel-Programme
            verschiedener Maschinen
